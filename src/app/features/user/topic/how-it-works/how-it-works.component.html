How Topic Submissions Work

  • Topics Go Live Instantly

    Your topic will be added to the public list and open for support right away.

  • Map Visibility Depends on Support

    Only the most supported topics in each area appear on the map.

  • Limit of 3 Active Topics

    Most users can have up to 3 active topics. Upgrade to Premium to unlock more.

  • Your Name Is Public

    Your display name and username will be shown as the topic creator.
